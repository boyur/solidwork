# Фигурка самолёта для SolidWorks и 3D-печати

Простая пластиковая фигурка самолёта в двух вариантах. Обе модели
печатаются **без поддержек** (плоское основание) и не имеют мелких
хрупких элементов.

| Вариант | Файлы | Описание |
|---|---|---|
| **2D-силуэт** (самый простой) | `airplane_silhouette.dxf`, `.svg` | Чистый 2D-контур для тиснения/гравировки. Импортируется в эскиз SolidWorks |
| **Кубик с тиснением** | `airplane_stamp.swb` | Кубик 60 мм с оттиском самолёта глубиной 2 мм на грани |
| Плоский | `airplane_flat.stl`, `airplane_flat.swb` | Силуэт самолёта, пластина толщиной 4 мм. Печать ~15–20 мин |
| Объёмный | `airplane_figurine.stl`, `airplane_figurine.swb` | Фигурка с фюзеляжем, кабиной и килем, высота 34 мм |

Габариты силуэта: длина 120 мм, размах крыла 140 мм.
Масштаб легко меняется прямо в слайсере или при импорте.

## 2D-силуэт и тиснение на кубике

`airplane_silhouette.dxf` — единый замкнутый контур (22 точки).
Как использовать в SolidWorks:

1. **Файл → Открыть** → выбрать `.dxf` → «Импортировать в новую
   деталь» как 2D-эскиз, **или** в открытом эскизе:
   **Вставка → DXF/DWG**.
2. Дальше с контуром можно сделать что угодно: **Вытяжка** (плоская
   фигурка), **Вырез** на грани кубика (тиснение/штамп) или
   **Бобышка** на 1–2 мм (выпуклый рельеф).

Либо просто запустите макрос `airplane_stamp.swb` — он сам построит
кубик 60×60×60 мм и сделает на верхней грани оттиск самолёта
глубиной 2 мм (масштаб контура 0.4: длина 48 мм, размах 56 мм).
Контур генерирует скрипт `generate_silhouette_2d.py`.

## Как открыть в SolidWorks

Файлы `.swb` — это макросы, которые строят модель автоматически
обычными эскизами и вытяжками (всё параметрично, размеры можно
потом редактировать в дереве построения):

1. Откройте SolidWorks.
2. **Инструменты → Макрос → Выполнить…** (Tools → Macro → Run…).
3. Выберите `airplane_flat.swb` (или `airplane_figurine.swb`).
4. Макрос сам создаст новую деталь и построит самолёт.
5. Для печати: **Файл → Сохранить как → STL**.

Если какая-то вытяжка ушла вниз вместо вверх — откройте её в дереве
построения и нажмите «Реверс направления».

Готовые STL уже лежат в репозитории — можно печатать сразу, без
SolidWorks. Их генерирует скрипт `generate_airplane_stl.py`
(чистый Python, без зависимостей): `python3 generate_airplane_stl.py`.

## Рекомендации для печати (PLA)

- Сопло 0.4 мм, слой 0.2 мм
- Заполнение 15 %
- Поддержки: **не нужны**
- Кайма (brim): не обязательна; для плоской версии можно добавить
  5 мм, если стол прилипает плохо
- Ориентация: как в файле — плоской стороной на стол

В слайсере детали модели слегка перекрываются — это нормально,
Cura/PrusaSlicer автоматически объединяют их в одно тело.
